A feladat leírása:

Numerikus integrálás elvégzése az x^2 függvényen a trapéz módszerrel A=-15 és B=15 korlátok között, EPS pontossággal.
Az EPS csökkentésével érem el a feladat bonyolultságának a növekedését.

Vizsgált esetek:
    -Sequential
    -2 Threads
    -4 Threads
    -8 Threads
    -16 Threads
